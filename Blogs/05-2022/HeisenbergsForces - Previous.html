Heisenberg's Mysterious Forces

One particle quantum systems: quick base knowledge

Though Quantum mechanics in one dimension is relatively straightforward, we can find a couple of exciting surprises here and there. Let's start with one particle.

Where $\psi$ comes from

Through Schrödinger's equation, we can solve our system for a given potential. More specifically, the "complete" version of Schrödinger's equation is reduced to a time-independent equation:

\[-\frac{\hbar ^2 }{2m}\frac{d^2 \psi }{dx^2 }+V\psi =E\psi \;\;\Leftrightarrow \;\;\widehat{H}\psi =E\psi \]

From this, we can find the wave functions $\psi$ that describe our system. Classic problems include:

  • The particle in a box:
    • \[V(x)= \begin{cases} \begin{matrix} 0 & 0 < x < L \\ \infty & \text{elsewhere} \\ \end{matrix} \end{cases} \;\;\;\Rightarrow \;\;\;\psi _n(x)=\sqrt{\frac{2}{L}}\sin \left( \frac{n\pi }{L}x \right) \]
  • The simple harmonic oscillator:
    • \[V(x)=\frac{1}{2}m\omega ^2 x^2 \;\;\;\Rightarrow \;\;\; \psi _n(x)=A_n\text{H} _n \left[ \sqrt{\frac{m\omega }{\hbar }}x \right] \exp \left( -\frac{m\omega }{2\hbar }x^2 \right) \]

In both examples, $n$ denotes the energy level. For particles in a box, $n$ starts from $1$. For the simple harmonic oscillator, we start at $0$.

Either way, $\psi $ describes what we call probability amplitude which then becomes a probability density after we take the squared modulus.

\[|\psi |^2 =\psi ^* \psi \tag{Yes, $\psi $ can be complex!}\]

This is great because now we can use integrals to calculate probabilities to find a particle between one point and another one:

\[\int_{0}^{1} |\psi |^2 dx=\text{some value between 0 and 1.} \]

Of course, when we take the integral over all space, we should expect it to yield $1$.

\[\int_{-\infty }^{\infty } |\psi |^2 dx=1 \]
